# 🚀 手动部署指南（无需Git）

## 📋 部署前准备

### 1. 注册GitHub账号
1. 访问 https://github.com
2. 点击"Sign up"注册账号
3. 填写用户名、邮箱、密码
4. 验证邮箱

### 2. 创建仓库
1. 登录GitHub后，点击右上角"+"号
2. 选择"New repository"
3. 仓库名填写：`snake-game`
4. 选择"Public"（公开）
5. 不要勾选任何选项
6. 点击"Create repository"

## 📁 上传游戏文件

### 方法一：网页上传（推荐）

1. **进入仓库页面**
   - 创建仓库后会自动进入仓库页面
   - 或者访问：`https://github.com/你的用户名/snake-game`

2. **上传文件**
   - 点击"uploading an existing file"
   - 将以下文件拖拽到上传区域：
     - `index.html`
     - `style.css`
     - `script.js`
     - `README.md`
     - `DEPLOY.md`
     - `分享方案.md`
     - `手机访问指南.md`
     - `游戏截图.md`
     - `ICONS.md`
     - `package.json`

3. **提交文件**
   - 在页面底部填写提交信息：`Initial commit: 贪吃蛇游戏`
   - 点击"Commit changes"

## 🌐 启用GitHub Pages

1. **进入设置页面**
   - 在仓库页面点击"Settings"标签

2. **配置Pages**
   - 左侧菜单选择"Pages"
   - Source选择"Deploy from a branch"
   - Branch选择"main"
   - 点击"Save"

3. **等待部署**
   - 几分钟后，页面会显示你的网站链接
   - 链接格式：`https://你的用户名.github.io/snake-game/`

## 📱 测试访问

1. **电脑测试**
   - 在浏览器中打开你的网站链接
   - 确认游戏可以正常运行

2. **手机测试**
   - 在手机浏览器中打开同样的链接
   - 确认触摸控制按钮正常显示
   - 测试游戏功能
